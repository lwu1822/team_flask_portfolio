About the Project

In this project, we are making a smaller version of an online dictionary. Simply enter the word and the definition will show up, including the type of word, examples used in context, and the pronunciation (just like a real dictionary!). We also chose to a Word of the Day to give the project somehting interesting

How is This Done

The point of the project was to learn about using APIs. We used an API that contained the words and their definitons in the backend of our project The project mainly uses HTML, Python, and Javascript


API used from rapidapi.com
